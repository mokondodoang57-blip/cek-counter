🔥 Cara Aktifkan Sync Firebase (Gratis 2 Menit)

×
Langkah 1: Buka console.firebase.google.com → Add Project → nama: cek-counter → Continue
Langkah 2: Build → Firestore Database → Create Database → Start in Test Mode → Lokasi asia-southeast2
Langkah 3: Project Settings (ikon gerigi) → Your Apps → Web → Register → Copy config seperti ini:
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "...",
  projectId: "...",
  ...
};
Langkah 4: Buka file index.html ini di Notepad, cari baris 24-31, ganti GANTI_API_KEY dll dengan config kamu, save, upload ulang ke GitHub ganti index.html lama.

Setelah aktif: Semua HP tim yang login akan sinkron otomatis. Kalau tim input, di HP kamu langsung muncul tanpa refresh!
